# [Combined Montreal Metro and REM Status Page](https://metro-rem-status.vercel.app)

A small Next.js app that shows service status for the four metro lines and the REM together on one page.

## Getting Started

Clone the repository:

```sh
git clone https://github.com/darcymccoy/metro-rem-status.git
```

Install the dependencies:

```sh
npm install
```

To run the development version locally:

```sh
npm run dev
```

This will be hosted at [http://localhost:3000](http://localhost:3000).

### Production Build

To create a production build:

```sh
npm run build
```

And to run that production build:

```sh
npm run start
```
